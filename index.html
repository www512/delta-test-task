Пояснение к заданию: Таблица формируется динамически при загрузке данных с сервера в формате JSON. Значения для графиков генерируются случайным образом. Для построения графиков используется библиотека Chart.js. При положительном проценте ячейка и текст автоматически закрашиваются в зелёный цвет, при отрицательном в красный, при нуле не изменяются.
